* Iniciando/subindo o Mongo
	no terminal/console digite o comando "mongod"
	
* Exibindo as bases de dados no mongodb
	show databases;
	
* Acessando uma base de dados mongodb
	use databasename;
	
* Exibindo as coleções de uma base mongodb
	show collections;

* Exibindo os registros de uma coleção (Coleção chamada usuarios)
	db.usuarios.find();

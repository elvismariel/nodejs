* Inicia npm project dependency
	npm init --yes
	
* Instalação do Express
	npm install -s express
	
* Instalação do Sequelize, ORM que mapeia o banco de dados + Modulo Mysql
	npm install --save sequelize
	
		- Modulos
			npm install --save mysql2
			npm install --save mariadb
			
* Instalação do Handlebars, ferramente de auxilio a criação de páginas html(template)
	npm install --save express-handlebars
	
* Instalação do body-parser, utilitário que auxilia o recebimento de dados de formulário.
	npm install --save body-parser
	
* Instalação do Mongoose, ODM que mapeia os documentos do Mongo
	npm install --save mongoose

* Instalação do Mongoose-paginate, paginação nos documentos do Mongo.
    npm install mongoose-paginate
	
* Instalação do Express Session para trabalhar com sessão na aplicação
	npm install --save express-session
	
* Instalação do Connect Flash
	npm install --save connect-flash
	
* Instalação do bcryptjs, biblioteca para encriptar senha/dados
	npm install --save bcryptjs
	
* Instalação do passport, biblioteca de autentificacao diversos tipos (linkedin, facebook...)
	npm install --save passport
	npm install --save passport-local (estrategia escolhida para login local)

* Instalação do plugin para gerenciar os diretorios
    npm install require-dir

* Instalação do plugin cors para liberar o acesso de outros dominios, nao somente do localhost
    npm install cors

* Instalação do create-react-app
    npm install -g create-react-app

-------------------------- Erros --------------------------
* npm ERR! code ELIFECYCLE
    Resolver:
        - Step 1: $ npm cache clean --force
        - Step 2: Delete node_modules by $ rm -rf node_modules
        - Step 3: npm install
